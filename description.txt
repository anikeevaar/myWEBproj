Эко-система Essentia: сервисы Harmony Dairy и PayMind

1. Введение
Essentia — это цифровая экосистема, объединяющая два сервиса для улучшения качества жизни:

Harmony Dairy — дневник настроения с анализом психического здоровья.

PayMind — система контроля подписок и финансовых расходов.

Оба сервиса интегрированы с Telegram-ботами для удобства пользователей.

2. Сервис Harmony Dairy
2.1. Назначение
Помогает пользователям отслеживать эмоциональное состояние, формировать полезные привычки и получать психологическую поддержку.

2.2. Функционал
Вопрос дня: Ежедневные персональные вопросы для рефлексии.

Дневник настроения: Запись эмоций.

Аналитика: Графики и статистика по настроению за неделю/месяц.

Telegram-бот:
Связь с разработчиками.
2.3. Технологии
Backend: Python (Flask), SQLite.

Telegram Bot: python-telegram-bot.

Аналитика: Pandas.

3. Сервис PayMind
3.1. Назначение
Контроль подписок и платежей для предотвращения ненужных трат.

3.2. Функционал
Добавление подписок: Название, стоимость, дата оплаты, ссылка.

Уведомления:

Telegram-бот присылает напоминание за 1 день до списания.

Ежемесячный отчет о расходах.

Автопродление: Отметка об оплате для сброса статуса.

3.3. Технологии
Backend: Python (Flask), SQLite.

Планировщик: APScheduler для уведомлений.

Telegram Bot: Уведомление о предстоящих платежах.

4. Интеграция и безопасность
Аутентификация: логин/пароль + Telegram ID.

Шифрование: Хеширование паролей (werkzeug.security).

5. Целевая аудитория
Harmony Dairy:

Люди с высоким уровнем стресса.

Пользователи, интересующиеся саморазвитием.

PayMind:

Пользователи с 3+ активными подписками.

Те, кто хочет оптимизировать расходы.

6. Перспективы развития
Мобильное приложение (React Native).

ИИ-аналитика в Harmony Dairy (прогнозирование настроения).

Интеграция с банками для PayMind (автосбор данных о подписках).

7. Заключение
Essentia сочетает заботу о ментальном здоровье (Harmony Dairy) и финансовую осознанность (PayMind). Использование Telegram-ботов делает сервисы удобными для ежедневного использования.
